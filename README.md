Brings Star Wars to minetest ! Do /team <either jedi or sith> to choose your faction .

--Jedi Abilities

Dash : Sneak + Up

Jump : Sneak + Jump

Push : Sneak + Place/Use (while looking at a player)

Saber Throw : Sneak + Place/Use (while holding an open lightsaber)

Heal : Sneak + Place/Use

Force Sense : Sneak + Place/Use

Crystal Bond : Sneak + Place/Use(while holding a blank kyber crystal)

--Sith Abilities 

Dash : Sneak + Up

Jump : Sneak + Jump

Pull : Sneak + Place/Use (while looking at a player)

Saber Throw : Sneak + Place/Use (while holding an open lightsaber)

Choke : Sneak + Place/Use(while looking at a player)

Force Lightning : Sneak + Place/Use(while looking at a player)

Crystal Bleed : Sneak + Place/Use(while holding a blank kyber crystal)

You can choose which ability to use via aux1 + dig/place keybind (default E + Left Click). You can see which ability and team you have chosen at the left down corner.
